The `alert` structure has some classes for different status:

.ui-alert-read: changes the text color for alerts readed.
.ui-alert-sm & .ui-alert-xs: applies the media queries css rules in its related width as a css classes.
.ui-alert-icon & .ui-alert-icon_show: hide/show for the alert icon at left.
.ui-alert-text_show: shows the whole alert text without height restrictions.
.ui-alert-status-text & .ui-alert-status-text_show: hide/show for the status text, above the alert main button.
.ui-alert-hidden-text-btn & .uui-alert-hidden-text-btn_show: hide/show the button related with "view more" action, at alert text.
