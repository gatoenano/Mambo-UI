The `header` structure has some classes for different status:

.ui-header-global-section: changes the borders to the global section border color.
.ui-header-sm & .ui-header-xs: applies the media queries css rules in its related width as a css classes.
.ui-header-notification_active: shows the notification rectangle.
.ui-tooltip-arrow-top_left: puts the arrow at left of the tooltip container.
.ui-tooltip-arrow-top_right: puts the arrow at right of the tooltip container.
