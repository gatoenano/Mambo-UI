Global utilities

Global utilities are designed to reduce the frequency of highly repetitive declarations in CSS while allowing for quick and easy development.

Spacing

Assign margin or padding to an element or a subset of its sides with shorthand classes. The classes are named using the format: {property}-{sides}-{size} (size 3rem max.)

Here are some representative examples of these classes:

.m-t-0: margin - top - 0

.m-b-1: margin - bottom - 1

.m-x-auto: margin - left&right - atuo

.p-l-2: padding - left - 2

.p-r-3: padding - right - 3

.p-a-3: padding - all - 3

Text transform

Transform text with text capitalization classes.

.text-lowercase: Lowercased text.

.text-uppercase: Uppercased text.

.text-capitalize: CapiTaliZed text.

Font weight and italics

Quickly change the weight (boldness) or italicize text.

.font-weight-bold: Bold text.

.font-weight-normal: Italicized text.

.font-italic: Italicized text.
