--- notes: |

The `select` component has two classes for choose the position of the label and one more for error status:

.ui-select-label-text: shows the label above the select.
.ui-select-label-text_left: shows the label at left of the select.
.ui-select_error: highlights an error showing the label and select in red.
With the :disabled property, the select element appears with disabled style.
