The `svg icons sprite` is used through an svg markup with css classes. Every icon can be changed with css in many ways as needed:

The more common css properties for modification of colors & borders are:

  • fill: defines the color of the shape inside its outline. For example fill: '#3333ff';
  • stroke: defines the outline of the shape. For example stroke: '#00ff00';

There are many properties for svg manipulation; fill & stroke has many 'sub-properties' to explore, so it's important to have a good reference of that.

For changes related to the size, width & height css properties are enough.
